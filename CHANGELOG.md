# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-29

### Added

- Excalidraw skill definition (SKILL.md) with 4-step workflow: Design, Generate JSON, Render, Review
- Headless PNG/SVG renderer via Playwright + Excalidraw 0.17.6
- Bundled fonts: Virgil.ttf (hand-drawn) and Excalifont.ttf
- One-command installer (`install.sh`) with multi-package-manager support
- Monochrome-first design philosophy with optional accent color
- Configurable render options: width, height, scale, theme (light/dark)
- Bilingual documentation (English + Chinese)
- Example diagrams: architecture layers, workflow
